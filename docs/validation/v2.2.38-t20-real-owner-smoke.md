## 5. Test-side Finding

v2.2.37 resolved the following test-side issues discovered during T20:

- Git cannot preserve normal file `0600` mode across `git clone`.
- Safe fake credential fixtures need `chmod 600` at test runtime.
- Old BSD-first `stat` usage (`stat -f '%Lp'`) failed on GNU/Linux, producing filesystem info instead of a permission number.
- v2.2.37 corrected the test-side behavior by reversing the probe order to try GNU `stat -c '%a'` first.
- Business code was not changed.
