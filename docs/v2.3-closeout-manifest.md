## Safety Model

- **Read-only by default**: All planning/preflight commands do not mutate DNS, certificates, tokens, or Workers.
- **Exact confirmation phrases**: Dangerous actions require explicit flags plus exact confirmation.
- **Production Worker protected**: `nanok`, `nanob`, `nanok.biankai314.uk`, `nanob.biankai314.uk` are blocked by default.
- **No Web/Bot behavior change**: Web login and Bot owner gate remain unchanged.
- **No release/tag in this stage**: Release requires explicit owner approval.
- **No raw secrets in output**: Tokens are masked, zone IDs are not printed, raw API responses are not exposed.
- **No service reload/restart**: No automatic service changes.
